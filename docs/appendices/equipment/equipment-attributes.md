#  Equipment Attribute Effects

## I.Q. Conversion
A certain amount of the character's I.Q. is added to their Attack Power.

| IQ     | +ATK       ]
|:------:|:----------:|
| 0-4    | 0          |
| 5-9    | 1          |
| 10-14  | 2          |
| 15-19  | 3          |
| 20-24  | 4          |
| 25-29  | 5          |
| 30-34  | 6          |
| 35-39  | 7          |
| 40-44  | 8          |
| 45-49  | 9          |
| 50-54  | 10         |
| ...    | ...        |
| X      | floor(X/5) |


## I.Q. / Piety Barrier 
A certain amount of the character's I.Q / Piety is added to their Defense Power.

### Shields:
e.g., Mage Shield, Defender of Faith

| IQ/PIE  | +DEF        ]
|:-------:|:-----------:|
| 0-9     | 0           |
| 10-19   | 1           |
| 20-29   | 2           |
| 30-39   | 3           |
| 40-49   | 4           |
| 50-59   | 5           |
| 60-69   | 6           |
| 70-79   | 7           |
| 80-89   | 8           |
| 90-99   | 9           |
| 100-109 | 10          |
| ...     | ...         |
| X       | floor(X/10) |
