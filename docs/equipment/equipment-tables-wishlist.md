# Equipment Tables Wishlist

The tables below show a live view of equipment items missing data in the  [Wizardry Variants Equipment Table (editable)](https://docs.google.com/spreadsheets/d/1j9gmngMUitaMAESlcCNeWJ5vAdMo_-yo8KkPxiMr1ZY/edit), which is a publicly editable Google Sheet. 

Items are grouped by type and sorted by subcategory. Missing enhancement levels follow the name in (). If there are a bunch of blank lines, it's because new items were added and the source spreadsheet wasn't manually resorted.  If you have missing items or stats, please consider contributing screenshots in the [Discord equipment contributions channel](https://discord.com/channels/1296602475918524507/1443698753864208384).

If the table does not display correctly, try the [embedded version](./equipment-tables-embed.md).  The Wishlists are the first two tabs after the Instructions page.
