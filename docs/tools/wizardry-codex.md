# Wizardry Codex

A comprehensive companion tool for Wizardry Variants: Daphne — featuring an interactive Well of the Mind planner, party builder, adventurer manager, skill encyclopedia, and more.

**[Launch Wizardry Codex](https://wizardry-codex.pages.dev/)** 

---

## Features

### Well of the Mind Planner

Interactive trait tree planner. Spend Mind Pearls to unlock nodes, choose between multi-trait options, and see your build summary in real time.

- Click a node to spend pearls, click again to refund (dependent nodes are removed too)
- Multi-trait nodes let you pick which power to unlock
- Track **Guiding Lights** — see how many pearls you've earned from story progress
- Share your build via a one-click link

### Party Builder

Assemble your ideal 6-member party with a drag-and-drop interface.

- Place adventurers in the front or back line
- Filter by element (Fire, Water, Earth, Air, Light, Dark, Void) and class (Fighter, Knight, Mage, Priest, Thief, Ninja, Samurai, Others)
- Sort by list order or tier ranking
- Toggle adventurers on/off — inactive ones are hidden from the pool
- TODO - add team power list and activation rules check validation for alignment and element compatibility



### Adventurer Manager

Browse all 70 adventurers across Legendary, General, and Anonymous rarities. Filter by element and class, or switch between list and tier view.

### Skills Encyclopedia

Browse all 320+ skills in progression or list view. Filter by inheritance type (Inheritable, Unique, Discipline), class skills, and spells.

### Additional Encyclopedias

- **Monster Encyclopedia** — search and filter monsters by race, element, or category
- **Item Encyclopedia** — browse items by category
- **Relic Brews** — look up relic powers by star rating, stat, or alignment

### Abyss Quests & Wheel Flow

Track quests across all abysses with walkthroughs and reward information. Filter by side quests, faction quests, and more.
